# 智识游侠 - AI 学习助手

一个基于 RAG 和 AI 的智能学习平台，支持联网搜索和向量检索。

## 快速部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/interstellaris-hi/----&framework=fastapi)

## 本地运行

```bash
pip install -r requirements.txt
export API=your_api_key
python main.py
```

访问 http://127.0.0.1:8001

## 功能

- 📚 知识库管理 - 上传和向量化教材
- 💬 AI 对话 - 智能问答助手
- 🌐 联网搜索 - 本地资料不足时自动联网
- 📝 考试中心 - 生成和批改测验
